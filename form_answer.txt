I picked a small problem I face regularly: my downloads folder becomes messy with PDFs, screenshots, code files, and internship documents mixed together. I built a Python script that automatically scans a folder and organizes files into categories like Images, Documents, Code, Videos, and Others based on file extensions. I used ChatGPT to generate the initial sorting logic and help debug some folder path issues, but I manually customized the categories, tested the script, and verified whether files were moving correctly. The main thing working right now is automatic file organization and folder creation. One limitation is that it currently depends only on file extensions, so it can still misclassify unusual files. If I continue improving it, I would add AI-based file understanding and automatic renaming.
